# GDevelop 5

GDevelop 是一个开源、跨平台的**游戏编写器**，其设计为为任何人可用 - 无需任何编程技巧。如您从未使用过GDevelop, [请在线试用](https://4ian.github.io/GD)或者[下载桌面版](http://gdevelop-app.com)!

此**维基百科**旨在教导您如何使用本软件，并指导您入门。同样我们欢迎您对其作出贡献！

# 入门
